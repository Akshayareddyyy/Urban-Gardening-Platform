## Overview
A comprehensive digital platform for urban gardening enthusiasts to plan, track, and manage their urban gardens. This project provides tools for plant selection, garden layout planning, watering schedules, and harvest tracking.

## Features
- **Garden Planner**: Visual layout tool for designing urban gardens
- **Plant Database**: Comprehensive information about plants suitable for urban gardening
- **Watering Scheduler**: Automated reminders for plant care
- **Growth Tracker**: Monitor and log plant progress
- **Harvest Calendar**: Track when plants are ready for harvest
- **Resource Library**: Guides, tips, and best practices

## Getting Started
1. Open `src/html/index.html` in your web browser
2. Explore the plant database to select suitable plants
3. Use the garden planner to design your space
4. Set up watering schedules for your plants
5. Track your garden's progress over time
